# Chess-for-Nerds
Play chess versus your terminal buddy

### How to Play

On your favorite terminal paste the following:
```
git clone https://github.com/DevPranjal/Chess-for-Nerds.git
cd Chess-for-Nerds/chess-PvP/
python play.py
```
